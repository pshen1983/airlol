Register a Lotusy Account

Description
Registers an user to Lotusy accounts and get access token for further requests.


Method
POST


URL
/register/{type}

(Note: type needs to be "facebook" or "weibo")


Header
NameValue
Content-Typeapplication/json
app-key{your_unique_application_key}
