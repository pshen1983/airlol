Forget Password API

Description
API to trigger an forget password email to be sent to email provided in the request body.


Method
POST


URL
/account/email/forget




Header
NameValue
Content-Typeapplication/json



Request Body
{
"email":"test001@cairyme.com",
"recaptcha":"ccia68ngaerh3biuvhyser"
}



Response
CaseBody
SUCCESS 200 OK

{
"status":"success"
}
ERROR 400 Bad Request

{
"status":-1,
"description":"empty_email"
}
ERROR 400 Bad Request

{
"status":-1,
"description":"empty_recaptcha"
}
ERROR 400 Bad Request

{
"status":-1,
"description":"invalid_email_format"
}
ERROR 403 Forbidden

{
"status":-1,
"description":"not_authorized"
}
ERROR 500 Internal Server Error

{
"status":-1,
"description":"internal_server_error"
