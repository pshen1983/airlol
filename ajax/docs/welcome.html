CairyMe AJAX Services
HOST - <https://{env}.cairyme.com/ajax>


CairyMe AJAX contains following categories:
  • Account
  • Trip
  • Good
  • Message

For more details about each AJAX, please click on the AJAX link on the left.

AJAX detail contains:
  • What the AJAX does. (e.g. Process a Credit Card Online Payment)
  • AJAX method. (e.g. POST)
  • AJAX uri. (e.g. /web/payment/{user_id}/creditcard)
  • Request body information (e.g. {"save_card":true,"order":{"order_id":"13fasiudbvahserhawer"}})
  • Request response success and failed messages
