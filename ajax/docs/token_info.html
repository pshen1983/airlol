Access Token Information

Description
Get an access token's information.


Method
GET


URL
/tokeninfo?access_token={access_token}


Header
NameValue
Content-Typeapplication/json
app-key{your_unique_application_key}



Request Body
GET parameter {access_token}


Response
CaseBody
SUCCESS 200 OK

{
"status":"success",
"user_id":13,
"token_type":"Bearer",
"refresh_token":"",
"expires_in":14125123
}
ERROR 400 Bad Request

{
"status":"error",
"description":"missing_access_token"
}
ERROR 404 Not Found

{
"status":"error",
"description":"access_token_not_found"
}
